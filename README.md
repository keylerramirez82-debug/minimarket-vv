# Minimarcket V&V

Repositorio del proyecto web "Minimarcket V&V".

## Descripción

Sitio web estático para un minimarket con navegación de productos, sección de ofertas y carrito de compras.

## Contenido

- `index.html`: página principal
- `style.css`: estilos globales
- `cart.js`: lógica del carrito de compras
- `productos_de_limpieza.html`, `snack.html`, `frutas.html`, `bebidas.html`, `alimentos.html`, `bebidas_alcoholicas.html`: páginas de categorías
- `login.html`, `register.html`: autenticación
- `Delivery.html`: información de entrega
- `opiniones.html`: opiniones de clientes
- `imagenes/`, `iconos/`: archivos multimedia

## Uso

1. Abre `index.html` en el navegador.
2. Navega entre categorías y productos.
3. Usa el botón del carrito para ver la ventana modal del carrito.
